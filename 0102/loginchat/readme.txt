# 星型拓扑结构通信网络 #
使用方法：
1-编译host作为服务器，传入FIFO文件参数login；
2-编译node.c作为子端，复制若干个子端（目前设定上限100个，可修改host中的define N）
3-子端传入一个管道文件作为参数，子端用户名即管道文件名
4-运行。可随时上下线，保证通信的畅通。

————————说明————————
采取了动态分配端口的方法。
用户携带用户名和同名管道上线，host接收从login管道传递的登录信息
从预分配的数组空间中寻找一个为0的分配给该用户。

在host中采用变量active作为计数来统计当前在线用户数。
基本功能和之前不变。

node端是host端的简化版本，如果要使全部节点都具有服务器的功能，
可以修改函数对argv参数的调用来实现
